# 🤖 Pax26: The Next-Gen AI Automation Engine

**Pax26** is a premium AI-driven automation platform designed to revolutionize how businesses interact with customers on WhatsApp. By combining the official Meta Cloud API with advanced AI training, Pax26 enables businesses to scale their operations with intelligent, 24/7 automated agents that sound human and drive results.

![Pax26 Hero](public/Screenshot-chip.png)

## 🚀 Core Automation Features

### 🧠 AI Agent Training
Train a sophisticated AI agent that understands your unique business tone, services, and FAQs. Feed it your data and watch it handle complex customer queries with zero intervention.

### 📥 WhatsApp Smart Inbox (CRM)
A high-performance, immersive messaging hub built for focus.
*   **Real-time Sync**: Direct integration with Meta Cloud API.
*   **Handoff Control**: Toggle between "Smart Agent" and manual "Human Takeover" mode instantly.
*   **Lead Capture**: Automatically identify and save new prospects into your contact list.
*   **Anti-Ban Technology**: Built on official protocols to keep your business numbers safe.

### ⚡ Integrated Utility Services (Additional)
Pax26 also includes a fully integrated utility hub to manage essential payments directly within the ecosystem:
*   **Instant VTU**: Airtime and Data for all major networks.
*   **Utility Bills**: Pay Electricity and TV subscriptions (DSTV, GOtv, StarTimes).
*   **Wallet & Rewards**: Secure wallet with automated funding and integrated cashback on every transaction.

## 🎨 Design Philosophy
Pax26 is built with a **"Focus-First"** aesthetic:
*   **Glassmorphism UI**: A premium, modern look that rivals the best SaaS platforms in the world.
*   **App-within-an-App**: Immersive "Focus Mode" for high-volume tasks like the WhatsApp Inbox.
*   **Micro-animations**: Subtle interactions that make the automation experience feel alive.

## 🛠 Technical Architecture
*   **Framework**: Next.js 14+ (App Router).
*   **Core Logic**: Official Meta WhatsApp Cloud API.
*   **Intelligence**: Advanced AI integration with personalized business training.
*   **Database**: MongoDB & Mongoose.
*   **UI/UX**: Vanilla CSS for precision styling, Framer Motion for fluidity.

## 📦 Getting Started

### Installation
```bash
git clone https://github.com/empire688682/Pax26.git
cd Pax26
npm install
```

### Environment Configuration
Create a `.env.local` file:
```env
MONGODB_URI=your_mongodb_uri
NEXT_PUBLIC_META_APP_ID=your_id
NEXT_PUBLIC_META_EMBEDDED_SIGNUP_CONFIG_ID=your_config_id
```

### Launch
```bash
npm run dev
```
